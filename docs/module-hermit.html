Module: hermit

Provides classes for interacting with a Hermit installation through QR codes.

The base classes provided are HermitDisplayer and HermitReader for displaying & reading a QR code to/from Hermit, respectively. Functions parseHermitQRCodeData and encodeHermitQRCodeData handle the Hermit QR-code data protocol.

The following API classes are implemented:

  • HermitExportPublicKey
  • HermitExportExtendedPublicKey
  • HermitSignMultisigTransaction
